# Koiki UI

Sets of Koiki React Components

## Install

```
npm i -S koiki-ui
```

## Usage

```
import { Button, Input, InputtableButton } from 'koiki-ui';

const SampleComponent = () =>
  <div>
    <Button text="Search" />
  </div>;

```
